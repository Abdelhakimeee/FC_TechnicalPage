Introduction to HTML

HTML, or HyperText Markup Language, is the foundational language used to build web pages. HTML is used to create the basic structure of a webpage, including headings, paragraphs, and images.

HTML was first introduced in 1991 by Tim Berners-Lee, and it is now considered the backbone of all websites.

<!DOCTYPE html> <html>   <head>   </head>   <body>     <p>Welcome to HTML</p>   </body> </html>
  • Ease of use: HTML is easy to understand and learn.
  • Flexibility: It allows you to create interactive web pages.

    • Basic Structure of an HTML Page

    An HTML document consists of three main elements: <!DOCTYPE>, <html>, and <body>. The <!DOCTYPE> element indicates the type of document and defines the version of HTML being used.

    The <html> element contains all the elements of the page, while the <body> contains the actual content of the page that is displayed in the browser.

    <html>
      <head>...</head>
      <body>...</body>
    </html>
  • <!DOCTYPE>: Specifies the type of document.
  • <html>: Contains all HTML elements.
  • <body>: Contains the page content.

    • The <head> Element

    The <head> element contains metadata about the webpage, such as the title (which appears in the title bar), stylesheets, and meta information.

    This information helps enhance user experience and improve the page's ranking in search engines.

    <head>
      <title>My First HTML Page</title>
      <meta charset="UTF-8">
    </head>
  • Title: Specifies the title of the page.
  • Metadata: Provides additional information about the page.

    • The <body> Element

    The <body> element is where all the visible content of the web page is placed, including text, images, and interactive elements.

    Elements within the <body> include headings, paragraphs, links, tables, and lists.

    <body>
      <h1>Welcome to My Page</h1>
      <p>This is an example of HTML paragraphs.</p>
    </body>
  • Headings: Used to organize content.
  • Links: Used for navigation between pages.

    • Common Elements in HTML

    HTML includes many elements that are frequently used to create rich web pages. Some of these elements include <p> for paragraphs, <a> for links, and <img> for adding images.

    Common elements make it easy to organize and format content in various ways.

    <p>This is a text paragraph.</p>
    <a href="https://example.com">Click here</a>
    <img src="image.jpg" alt="Image">
  • Text Elements: <p>, <h1> - <h6>.
  • Interactive Elements: <a>, <button>.
  • Visual Elements: <img>, <video>.
